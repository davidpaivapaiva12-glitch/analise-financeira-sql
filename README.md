# 💰 Análise Financeira com SQL

## 📌 Sobre o Projeto
Este projeto simula um sistema de análise financeira, inspirado em operações bancárias.

## 🛠️ Tecnologias
- SQL Server

## 📊 Análises
- Saldo por cliente
- Entradas vs saídas
- Clientes com maior movimentação

## 🎯 Objetivo
Simular cenários reais de análise de dados financeiros.

## 🚀 Como executar
1. Rodar schema.sql
2. Rodar inserts.sql
3. Rodar queries.sql

## 💡 Insights
- Identificação de clientes com maior volume financeiro
- Análise de comportamento de gastos
